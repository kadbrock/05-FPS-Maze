# Skeleton Maze!
Completes Project 5 FPS Maze
April 13, 2023

Description
In this FPS Maze, you will navigate through it with the help of your minimap and gun, defeating skeletons along the way. You must find the key to exit.

## Implementation
I included:
- Multiple enemies
- Sound effects/Music
- Gun
- Decals
- Procedural generation
- Key
- Exit
